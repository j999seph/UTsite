# UTsite

UTsite is a one-page, premium, discreet business website for **Unique Trades**.  
It is designed to feel calm, intelligent, globally connected, and selective without over-explaining the scope of the work.

## Project Intent

- One long scrolling page with anchored navigation
- Minimalist layout with strong typography and strategic whitespace
- Dark theme with a premium, global-business feel
- Subtle motion only
- Built to encourage thoughtful, relationship-driven inquiries
- Reserved future sections for Network and Insights

## Stack

- Next.js
- Tailwind CSS
- Framer Motion
- TypeScript

## Local Development
